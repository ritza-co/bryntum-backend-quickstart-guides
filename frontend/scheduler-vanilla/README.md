# Vanilla JavaScript Bryntum Scheduler 

This is a basic Vite Bryntum Scheduler app.

## Getting started 

Start by following the guide to accessing the [Bryntum npm repository](https://bryntum.com/products/scheduler/docs/guide/Scheduler/npm-repository). Once youâ€™ve logged in to the registry, install the dependencies using the following command:

```shell
npm install
```

If you don't have a Bryntum Scheduler license, install the trial version:

```shell
npm install @bryntum/scheduler@npm:@bryntum/scheduler-trial
```

Run the local development server:

```shell
npm run dev
```
